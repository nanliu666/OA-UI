## List 列表

通用列表。

### 何时使用

最基础的列表展示，可承载文字、列表、图片、段落，常用于后台数据展示页面。

### 代码演示
:::demo 
```html
<template>
  <mg-list>
  </mg-list>
</template>

````
:::


### Attributes
| 参数       | 说明   | 类型  | 可选值 | 默认值 |
| ---------- | ------ | ----- | ------ | ------ |
| dataSource | 数据源 | any[] | —      | —      |


### Slot
| 参数   | 说明             |
| ------ | ---------------- |
| header | 当前列表头部插槽 |
| footer | 当前列表底部插槽 |

### Events
| 事件名称  | 说明               | 回调参数 |
| --------- | ------------------ | -------- |
| onConfirm | 点击确认按钮时触发 | —        |
